# 计数器 计数是一种最简单基本的运算，计数器就是实现这种运算的逻辑电路，计数器在数字系统中主要是对脉冲的个数进行计数，以实现测量、计数和控制的功能，同时兼有分频功能。 计数器在数字系统中应用广泛，如电子计算机的控制器中对指令地址进行计数，以便顺序取出下一条指令，在运算器中作乘法、除法运算时记下加法、减法次数，又如在数字仪器中对脉冲的计数等等。 ## 设计流程 ![描述](https://hatfniprpjrjwzmximna.supabase.co/storage/v1/object/public/notes-images/content-1770032146448.webp) ## 设计规划 ### 新建文件夹 在**FPGA_Pro**文件夹中新建**counter**文件夹，在**counter**文件夹中新建以下四个文件夹 ![描述](https://hatfniprpjrjwzmximna.supabase.co/storage/v1/object/public/notes-images/content-1770086172871.webp) ### 功能实现 实现一个具有计数器功能的电路，计数的时间间隔为1秒。 使用板载的led灯在前0.5秒点亮，后0.5秒熄灭。 ## 波形绘制 在**doc**文件夹下新建**visio**文件，绘制波形 ![image-20260214145053435](https://fastly.jsdelivr.net/gh/Handoly/web-site-images@main/imgimage-20260214145053435.png) ## 代码编写 在rtl文件夹中新建.v文件：先新建文本文档，命名为led修改后缀名为.v 双击打开.v文件，编写代码 ```verilog module counter #( parameter CNT_MAX = 25'd24_999_999 ) ( input wire sys_clk , input wire sys_rst_n , output reg led_out ); /*其他方式的参数定义 parameter CNT_MAX = 25'd24_999_999; localparam CNT_MAX = 25'd24_999_999; */ reg [24:0] cnt; always@(posedge sys_clk or negedge sys_rst_n) if(sys_rst_n == 1'b0) cnt <= 25'd0; else if(cnt == CNT_MAX) cnt <= 25'd0; /*其他方式的参数定义 parameter CNT_MAX = 25'd24_999_999; localparam CNT_MAX = 25'd24_999_999; */ reg [24:0] cnt; reg cnt_flag; always@(posedge sys_clk or negedge sys_rst_n) if(sys_rst_n == 1'b0) cnt <= 25'd0; else if(cnt == CNT_MAX) cnt <= 25'd0; else cnt <= cnt + 25'd1; always@(posedge sys_clk or negedge sys_rst_n) if(sys_rst_n == 1'b0) cnt_flag <= 1'b0; else if(cnt == CNT_MAX-25'd1) cnt_flag <= 1'b1; else cnt_flag <= 1'b0; */ endmodule ``` rtl视图如下： ![image-20260214154338967](https://fastly.jsdelivr.net/gh/Handoly/web-site-images@main/imgimage-20260214154338967.png) 进行仿真，波形如下： ![image-20260214154648073](https://fastly.jsdelivr.net/gh/Handoly/web-site-images@main/imgimage-20260214154648073.png)
